# 搭建 Static-First Next.js 官网基础壳

Label: `ready-for-agent`

## What to build

Build the first working shell of the Yingsheng Company Website using Static-First Next.js Architecture. The slice should create a crawlable multi-page foundation with shared layout, Primary Navigation, footer, responsive containers, base visual tone, route skeletons, and a Static Content Source pattern that later slices can extend.

## Acceptance criteria

- [ ] A Next.js, TypeScript, and Tailwind CSS project can run locally.
- [ ] The site renders a shared layout with Primary Navigation and footer.
- [ ] The navigation includes 首页, 核心业务, 服务案例, 映盛研究院, 映盛文化, 关于映盛, 加入我们, and FAQ.
- [ ] Route skeletons exist for all first-phase pages without broken navigation.
- [ ] Static content can be read from repository-managed content files.
- [ ] The visual foundation is professional, modern, restrained, and avoids pure technology style, flashy gradient style, and traditional corporate template styling.
- [ ] Basic responsive behavior works on desktop and mobile widths.

## Blocked by

None - can start immediately
