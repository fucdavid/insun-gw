# 实现首页首屏、六大业务摘要和咨询入口

Label: `ready-for-agent`

## What to build

Build the homepage as the main entry point for Potential Customers. It should communicate the Homepage Hero Message, explain Core Positioning, present the six Core Service Lines as Homepage Service Summaries, express automotive as the primary Advantage Industry while still naming other Target Industries, and guide visitors toward the Consultation CTA.

## Acceptance criteria

- [ ] Homepage H1 is “让品牌增长，回到真实用户关系”.
- [ ] Supporting copy explains Yingsheng's six Core Service Lines and long-term growth from public-domain acquisition to private-domain conversion.
- [ ] Primary CTA is “咨询合作”.
- [ ] Secondary CTAs include “查看核心业务” and “查看服务案例”.
- [ ] The homepage shows six Homepage Service Summaries with title, one value sentence, and inline service keywords.
- [ ] Homepage service summaries do not include representative client names.
- [ ] The homepage makes automotive visible as the primary Advantage Industry while also naming consumer goods, finance, tourism, and other Target Industries.
- [ ] The homepage is readable and usable on desktop and mobile.
